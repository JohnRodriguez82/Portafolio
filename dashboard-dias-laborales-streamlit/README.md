# 📊 Dashboard de Días Laborales – Streamlit

Dashboard analítico para calcular días laborales efectivos considerando
festivos, fines de semana y reglas de negocio por sección.

## 🚀 Tecnologías
- Python
- Streamlit
- Pandas / NumPy
- Altair

## ⚙️ Ejecución
```bash
pip install -r requirements.txt
streamlit run app.py
